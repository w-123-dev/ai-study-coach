# AI 考研私人教练 — 产品定位

## 一句话定位

> AI 考研私人教练：从今天到上岸那天，AI 陪你走完每一步。

## 产品使命

帮助考研学生完成从「目标设定 → 计划执行 → 每日反馈 → 动态调整」的**全周期备考闭环**，以 AI 长期陪伴的方式解决自律难、规划乱、心态崩三大核心痛点。

## 目标用户

| 用户类型 | 核心需求 | 使用场景 |
|---------|---------|---------|
| 自律差学生 | 需要外部监督和提醒 | 每日打卡、AI 教练督促 |
| 基础差学生 | 需要合理规划和信心 | 个性化计划、循序渐进任务 |
| 二战学生 | 需要针对性分析和调整 | 数据分析、弱项补强 |
| 焦虑型学生 | 需要情绪支持和方向感 | AI 聊天陪伴、学习进度可见性 |

## 核心差异（vs 竞品）

| 对比维度 | 传统学习 App | ChatGPT | AI 考研教练 |
|---------|------------|---------|------------|
| 计划生成 | 固定模板 | 一次性回答 | 个性定制 + 动态调整 |
| 长期记忆 | 无 | 会话级 | 用户画像 + 每日快照 + 会话摘要 |
| 主动提醒 | 无 | 等待提问 | 异常检测 + 主动教练消息 |
| 数据分析 | 简单统计 | 无 | 趋势分析 + 状态评估 |
| 教练人格 | 无 | 通用 | 4 种教练模式可选 |

## 用户价值层级

```
顶层：长期陪伴上岸（几百天持续使用）
  ↑
中层：动态调整计划（每周复盘 + AI 建议）
  ↑
基础层：每日任务执行（打卡 + 反馈）
  ↑
入口层：AI 生成计划（个性化备考路线）
```

## 产品边界（不做的事）

1. 不做课程内容 / 题库 / 视频教学
2. 不做心理咨询（仅学习陪伴）
3. 不做上岸承诺或结果保证
4. 不做社区 / 社交 / 排行榜
5. 不接入支付（暂不商业化）
